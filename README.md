### 📝 DESKRIPSI:
Script **hutang.sh** adalah solusi buat kalian yang sering lupa punya hutang atau piutang. Daripada ribet cari kertas atau buku, mending catat langsung di terminal. Cocok untuk memantau siapa yang belum bayar atau hutang mana yang belum lunas.

---

## 🐉 1. INSTALL DI KALI LINUX
```bash
sudo apt update && sudo apt install git bash nano -y
git clone https://github.com/abuesbol74-source/HUTANG.git
cd HUTANG
chmod +x hutang.sh
./hutang.sh
```

---

## 📱 2. INSTALL DI TERMUX
```bash
pkg update && pkg upgrade -y
pkg install git bash nano -y
git clone https://github.com/abuesbol74-source/HUTANG.git
cd HUTANG
chmod +x hutang.sh

```
